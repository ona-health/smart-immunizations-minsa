Esta página incluye los indicadores y métricas de desempeño que serían agregados a partir de los elementos de datos base, adaptados del [Kit de Adaptación Digital de la OMS para Inmunizaciones](https://iris.who.int/handle/10665/380303) al contexto del programa de inmunizaciones del Perú.

Para representaciones computables, consulte la página de <a href="indicators-measures.html">Indicadores y Medidas</a>.

Estos indicadores pueden ser agregados automáticamente desde la herramienta de seguimiento digital para alimentar el sistema de información de gestión en salud (HIS-MINSA).

### Descripción de columnas de la tabla de indicadores

| Elemento | Descripción |
|---|---|
| Código del indicador | Identificador del indicador |
| Nombre del indicador | Nombre corto del indicador |
| Descripción | Descripción del indicador |
| Definición del numerador | Descripción del numerador utilizado para calcular el indicador |
| Cálculo del numerador | Fórmula o método para derivar el numerador |
| Definición del denominador | Descripción del denominador utilizado para calcular el indicador |
| Cálculo del denominador | Fórmula o método para derivar el denominador |
| Desagregaciones | Desagregaciones disponibles para el análisis |
| Referencias | Fuentes de referencia del indicador |

### Indicadores de proceso

| Código | Nombre | Descripción |
|---|---|---|
| IMMZ.IND.20 | Tasa de deserción Penta1-Penta3 | Porcentaje de niños que recibieron la primera dosis de Pentavalente pero no completaron la tercera dosis. Fórmula: (Penta1 - Penta3) / Penta1 x 100 |
| IMMZ.IND.21 | Tasa de oportunidades perdidas | Porcentaje de visitas a establecimientos de salud donde un niño elegible para vacunación no recibió las vacunas correspondientes |
| IMMZ.IND.22 | Tasa de ESAFI notificados | Número de ESAFI notificados por cada 100,000 dosis administradas |
| IMMZ.IND.23 | Porcentaje de establecimientos con ruptura de stock | Porcentaje de establecimientos que reportaron desabastecimiento de al menos una vacuna del esquema durante el período |
| IMMZ.IND.24 | Completitud de registro | Porcentaje de registros de vacunación con todos los campos obligatorios completos en el sistema |
